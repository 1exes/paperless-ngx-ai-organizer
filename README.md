# paperless-ngx-ai-organizer

Automatische Dokumentenorganisation mit lokalem LLM (Ollama/LM Studio).

## Tech Stack
- Python
- Paperless-NGX
- Ollama / LM Studio (lokales LLM)
- Docker

## Features
- Automatische Klassifizierung von Dokumenten per KI
- Lokale Verarbeitung ohne Cloud-Abhaengigkeit
- Integration mit Paperless-NGX API

## Getting Started
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
